# BDFL Help

Run `bdfl help` as an ordinary tool action and present its output. Clarify the host-specific invocations when useful: `/bdfl:activate`, `/bdfl:list`, `/bdfl:help`, and `/bdfl:off` in Claude Code; `$bdfl:activate`, `$bdfl:list`, `$bdfl:help`, and `$bdfl:off` in Codex.

Do not activate BDFL, change state, broaden permissions, or choose a recovery action while showing help.
